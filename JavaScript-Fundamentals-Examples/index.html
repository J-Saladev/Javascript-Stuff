Example 5. Variables

Variables in JavaScript can be declared using: var, const and let keywords
  • var: Variables declared with var are global variables. Use var to keep compatibility with older browsers
  • const: Variables declared with const are global variables. As general rule, always declare variables with const
  • let: Variables declared with let are block-scoped variables. Use let if you think that the value of the variable can change
